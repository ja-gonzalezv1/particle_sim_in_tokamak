# Particle simulation in tokamak reactor
## functions
Some of the functions that are used for the other files.
## field_visualizer
Calculate and visualize the fields that are generated by the coils in a tokamak reactor. It is adviced to run this file as a Jupyter notebook instead, to make the calulations is one cell, and the visualizations in another.
## particle_simulation
Calculate and visualize the trajectory of a single particle in the helical field generated by the tokamak reactor.
## collisions
Simulate the collisions between multiple particles, calculate and visualize how their trajectories and velocities change with time.
